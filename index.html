//--Bachelor's Thesis--\\

PID Controller implementation in C++

In this project I'm programming a PID controller in C++ to run on Arduino UNO. The controlled circuit is a simple thermal one with resistor and thermistor configured as voltage divider to allow reading on voltage. The current from Arduino is amplified with UN2003A op-amp to warm up the resistor and consequently the thermistor. For correct modelling and control I use thermal calculations between objects.
