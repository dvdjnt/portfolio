SIMPLE BOARD GAME (JAVA)

In this project I programmed a simple board game as a part of my studies in OOP course at Slovak Technical University in Bratislava. This project implements inheritance, abstract classes and interfaces together in Factory and Decorator design patterns. The game consists of 3 types of figures (pawn, bishop, queen) in two colors (blue, red) racing towards a finish turn by turn, while being able to collect and equip various effects and items to increase their speed. Project template and unit tests were provided in this assignment as a way to standardize our solutions.
Availible for download on github

SENTENCE GENERATOR (PYTHON)

Random sentence generator in the style of infamous misinformation and conspiracy-spreading websites and figures of Slovakia. Program follows intricate conjugation rules of slovak language. Written in python, using OOP principles for possible scaling and using in other sentence generating projects. This was a fun little side project as a way to ease the tense socio-political situation in Slovakia with humour.

Availible for download on github
